Principais formatações

Negrito/Destaque

Nesas frase temos um termo em negrito usanto a tag <b>(não semântica)

Nesta frase, tenho um termo em destaque usando a tag <strong>

Itálico/Enfase

Nesta frase, temos um termo em itálico usando a tag <i>

Nesta frase, temos um termo em ênfase usando a tag <em>

Texto marcado

Podemos criar tambem um texto marcado usando a tag <mark>

Texto deletado

Podemos marcar um texto como excluido com a tag <del> que deve ser lido, masn= não considerado

Texto inserido

Podemos marcar um texto como inderido com a tag <ins> para dar uma enfase e indicar que ele foi adicionado depois.

Texto sobrescrito

Par inserir coisas do tipo x20+3, se usa a tag <sup>

Texto subscrito

Par inserir coisas do tipo H2O, se usa a tag <sub>
